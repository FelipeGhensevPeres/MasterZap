# 💬 MasterZap

O **MasterZap** é um chat web simples inspirado em aplicativos de mensagens, criado com **Flask, HTML, CSS e JavaScript**, onde um bot responde mensagens com base em uma base de conhecimento que ainda esta em desenvolvimento.

O objetivo do projeto é praticar **desenvolvimento web**, **integração entre frontend e backend** e **deploy de aplicações reais**, de forma simples.

## 🧠 Como o MasterZap funciona

- O backend é feito em **Flask**, responsável pela aplicação web  
- O frontend é construído com **HTML, CSS e JavaScript**  
- O bot funciona através de uma **base de conhecimento simples**, onde perguntas são comparadas e as respostas são retornadas automaticamente  
- O usuário digita uma mensagem e recebe a resposta do bot na própria interface
